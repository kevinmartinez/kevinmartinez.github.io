Uppgift B2 CSS Presentation

Uppgift B_2 går ut på att skapa en stilmall för webb-projektet www.csszengarden.com. CSS Zen Garden går ut på att visa världen vad CSS går för. Genom att alla som bidrar använder sig utav exakt samma HTML dokument, med en färdig struktur skrivet av teamet bakom CSS Zen Garden och bidrar med sina egna presentationer med hjälp av endast CSS.

Grymt rolig uppgift tänkte jag när den delades ut. Men vägen fram till en slutlig design blev lång. Några av mina första tankar och skisser var att försöka återskapa en trädgårdsliknande design, vilket jag snabbt kom på att det måste ha gjorts in absurdum med detta projekt. Mitt andra design / skissval gick ut på att försöka använda en underrubrik i strukturen som lyder: “The Road to Enlightment” och försöka skapa en sorts visuell väg med hjälp av bakgrundsbilder och egen grafik. Men jag fastnade här och beslutade mig för att köra hardcore CSS plugg med vår studieresurs Teamtreehouse.com.

Efter att ha gjort klart de flesta av deras kurser inom CSS förutom de mest avancerade så visste jag precis vad jag ville bidra med till the CSS Zen Garden. Nämligen en tutorial-liknande webbsida som beskriver de mest essentiella delarna av CSS med hjälp utav just CSS. Skissen är enkel, jag började med the box model och kände hur kul det var att arbeta med pseudo selektorer och pseudoelement för att kunna placera ut innehåll som inte finns i sidans HTML-struktur. Efter att ha kodat upp den första delen av skissen och insett hur galet mycket man kan göra så beslutade jag mig för att skissa upp CSS Floats för att kunna koda upp och visa hur det fungerar. Jag förstod att det skulle behövas många färger för att få fram budskapet så jag bestämde mig för att använda mig av flat colors, och använde mig av resurserna www.flatuicolors.com och www.flatuicolorpicker.com. Fördelen med de här färgerna var att de är tydliga och fungerar oftast väldigt bra att mixa huller om buller.

Jag känner att det här kan utvecklas väldigt mycket och då jag måste lämna in uppgiften så har jag bestämt mig för att fortsätta arbeta vidare på B_2 efter hand med att jag lär mig ännu fler CSS knep under utbildningens gång!
